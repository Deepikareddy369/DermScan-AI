# Skin Disease Classification using Deep Learning 🩺💻

## Overview 🌟

This project aims to classify different types of skin diseases using deep learning techniques,
specifically convolutional neural networks. It leverages a dataset containing images of
skin conditions and utilizes advanced models to achieve high accuracy in classification.
This system can assist doctors and individuals in early detection of skin diseases.

## Features ✨

- Efficient data preprocessing 🗂️
- Development of deep learning models using TensorFlow and Keras 🤖
- Performance evaluation with various metrics 📊
- User-friendly interface for predicting skin diseases from new images 📷
- Flexibility for customization and future enhancements 🚀

## Dataset 📁

The dataset used is sourced from a reliable repository, containing labeled images of various skin conditions. You can explore the data in the project's folders.

## Requirements ⚙️

Before starting, ensure you have the following installed:

- Python 3.8 or higher 🐍
- Required Python libraries:
  - tensorflow
  - keras
  - numpy
  - matplotlib
  - scikit-learn
  - opencv-python

Install all dependencies with:
```bash
pip install -r requirements.txt
```

## Installation 🚀

1. Clone the repository:
```bash
git clone https://github.com/Mohamedsayed01/Skin_disease_classification.git
cd Skin_disease_classification
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage 🔧

### Train the Model 🏋️‍♂️
Run the training script:
```bash
python train.py
```

### Evaluate the Model 📈
Evaluate model performance:
```bash
python evaluate.py
```

### Predict Skin Disease from a New Image 🩺
Predict using an image:
```bash
python predict.py --image path/to/your/image.jpg
```
*Replace `path/to/your/image.jpg` with your image path.*

## Results 🏆

🔸 Overall accuracy: **98%**  

Achieved an accuracy of up to 98%, demonstrating the model's effectiveness in skin disease classification with high precision.
