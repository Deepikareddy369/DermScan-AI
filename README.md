# 🩺 DermScan-AI

A deep learning-based skin disease classification system using MobileNetV2.

---

## 🚀 Features
- Classifies skin diseases from images
- Uses Transfer Learning (MobileNetV2)
- Supports multiple disease categories
- High accuracy model

---

## 🧠 Classes
- Acne and Rosacea
- Eczema
- Melanoma
- Psoriasis

---

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

---

## 📂 Project Structure
DermScan-AI/
│── Notebook/
│── model/
│── app.py
│── README.md


## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/your-username/DermScan-AI.git


2. Navigate to project folder:

cd DermScan-AI


3. Install dependencies:

pip install -r requirements.txt


4. Run the project:

python app.py


---

## 📊 Model Details
- Base Model: MobileNetV2
- Transfer Learning used
- Accuracy: ~98% (training accuracy)

---
